## 💡 Pro Tips

### 1. Label Best Practices

**Good Labels:**
- "Repeat Rows 1-4" ✅
- "Inc Every 6 Rows" ✅
- "Chart A" ✅
- "Dec Round" ✅

**Less Clear Labels:**
- "Counter" ❌ (too generic)
- "Thing" ❌ (unclear)
- "Rows" ❌ (what kind of rows?)

**Tip:** Be specific enough that you'll remember what it's for if you come back to the project after a break!

### 2. Multiple Counters Strategy

Use multiple counters on complex patterns:

**Example: Raglan Sweater**
```
Counter 1: "Body Inc Round" (8) - Track raglan increases
Counter 2: "Total Rounds" (50) - Track overall yoke length
Counter 3: "Rows Since Marker" (6) - Track between stitch markers
```

**Color-code them:**
- Blue = Main pattern repeats
- Green = Increases
- Red = Decreases
- Purple = Row/round counting
- Orange = Special sections

### 3. Target Number Selection

**Common targets by pattern type:**

| Pattern Type | Typical Target |
|--------------|----------------|
| Hat | 6-8 repeats |
| Sock | 8-12 (toe/heel) |
| Sweater raglan | 8-16 increases |
| Shawl | 12-30 increases |
| Scarf | 4-6 pattern repeats |
| Afghan square | 20-40 rows |

**Tip:** You can always edit the target later if the pattern changes!

### 4. When to Reset

**Reset when:**
- Starting a new pattern section
- Beginning the second sleeve (copy the target, reset count)
- Switching to a different repeat pattern
- Made a mistake and need to restart

**Don't reset:**
- If you just want to track total progress
- When checking your work (use decrement instead)

### 5. Counter Positioning

**Strategic placement:**
- Place near the relevant pattern section
- Keep off critical text/charts
- Group related counters together
- Use consistent positions across pages

**Example positions:**
```
Top-right: Main repeat counter
Top-left: Row/round counter  
Bottom-right: Special sections
Bottom-left: Notes/reminders
```

## 📱 Workflow Examples

### Example 1: Simple Hat

**Pattern says:** "Work crown decreases (k2tog, k8) around. Repeat decrease round every other round 6 times."

**Setup:**
1. Add counter
2. Label: "Crown Dec"
3. Target: 6
4. Place near the crown shaping instructions

**Use:**
- Work a decrease round → Tap counter (1/6)
- Work a plain round
- Work a decrease round → Tap counter (2/6)
- Continue until 6/6 (Complete!)

---

### Example 2: Raglan Sweater

**Pattern says:** "Increase 1 stitch each side of all 4 markers every other round 12 times."

**Setup:**
1. Add counter
2. Label: "Raglan Inc"
3. Target: 12
4. Place near yoke instructions

**Use:**
- Work increase round → Tap counter
- Work plain round
- Repeat
- When counter shows 12/12, raglan increases complete!

---

### Example 3: Lace Shawl

**Pattern says:** "Repeat chart 8 times, then work border chart 3 times."

**Setup:**
1. Add counter #1: "Main Chart" / Target: 8
2. Add counter #2: "Border" / Target: 3

**Use:**
- Work chart once → Tap "Main Chart" (1/8)
- Continue until main chart shows 8/8
- Start border → Tap "Border" counter
- When both are complete, shawl is done!

---

### Example 4: Complex Colorwork

**Pattern says:** "Work 24-row colorwork chart. Repeat 3 times total."

**Setup:**
1. Counter #1: "Chart Row" / Target: 24 (tracks position in chart)
2. Counter #2: "Chart Repeat" / Target: 3 (tracks how many times through)

**Use:**
- Complete row 1 → Tap "Chart Row" (1/24)
- Continue until 24/24
- Reset "Chart Row" to 0/24
- Tap "Chart Repeat" (1/3)
- Repeat process
- Done when "Chart Repeat" shows 3/3

## 🎨 Color Coding System

**Suggested color organization:**

### By Action Type
- 🔵 **Blue**: Neutral/mixed actions, pattern repeats
- 🟢 **Green**: Increases, growth sections
- 🔴 **Red**: Decreases, shaping down
- 🟣 **Purple**: Stitch patterns, cables, lace
- 🟠 **Orange**: Row/round counting, measurements

### By Project Section
- 🔵 **Blue**: Body/main section
- 🟢 **Green**: Sleeves
- 🔴 **Red**: Neckline/collar
- 🟣 **Purple**: Edging/borders
- 🟠 **Orange**: Special features

### By Priority
- 🔴 **Red**: Critical (must complete before next step)
- 🟠 **Orange**: Important (should track)
- 🔵 **Blue**: Normal (helpful but optional)
- 🟢 **Green**: Completed sections
- 🟣 **Purple**: Nice-to-have tracking

**Tip:** Pick one system and stick with it across all your projects!

| Action | How To |
|--------|--------|
| **Increment** | Single tap on counter |
| **Decrement** | Long press → "Decrement (-1)" |
| **Reset** | Long press → "Reset to 0" |
| **Edit label** | Long press → "Edit" → change label |
| **Change target** | Long press → "Edit" → use stepper |

## 🎯 Project Setup Checklist

Starting a new knitting project? Set up your counters efficiently:

- [ ] Read through entire pattern first
- [ ] Identify all sections that need repeat tracking
- [ ] Create counters for each section (but don't overdo it!)
- [ ] Label clearly and specifically
- [ ] Choose appropriate target counts
- [ ] Color-code logically
- [ ] Position near relevant instructions
- [ ] Test increment/decrement before starting
- [ ] Start knitting!

## 🔧 Troubleshooting

### "I accidentally tapped too many times!"
→ Long press → "Decrement (-1)" to go back

### "I need to frog back several rows"
→ Long press → "Edit" → use minus button to adjust count quickly

### "I finished but the pattern changed"
→ Long press → "Edit" → change target number → continue incrementing

### "I want to track the same thing on the second sleeve"
→ Create new counter, copy the label/target, position on second sleeve instructions

### "Counter is in the way"
→ Drag it to a new location

### "Wrong color"
→ Long press → "Edit" → pick new color

### "Need to start this section over"
→ Long press → "Reset to 0"

## 📖 Integration with Patterns

### Reading Pattern Instructions

When you see phrases like:
- **"Repeat X times"** → Create counter with target = X
- **"Work every Nth row/round"** → Create counter to track the repetitions
- **"Until piece measures..."** → Create row counter to estimate
- **"Increase/decrease X times"** → Counter for X

### Smart Counter Naming

Include key info in the label:
- What: "Inc" / "Dec" / "Repeat" / "Chart"
- Where: "Sleeve" / "Yoke" / "Rows 1-4"
- How often: "Every 6 Rows" / "Every Round"

**Examples:**
- "Inc Every 6 Rows" (what + how often)
- "Chart A Repeat" (what + where)
- "Sleeve Dec Round" (where + what)
- "Rows 1-4 x6" (what + target)
